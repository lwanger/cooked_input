.. currentmodule:: cooked_input

GetInput
********

The :class:`GetInput` class is the heart of the ``cooked_input`` library. Calls to :class:`GetInput` objects
perform the cleaning, conversion and validation of input data.

.. note::

 Using the :class:`GetInput` class is for more advanced users, Beginners can just use
 the `convenience functions <get_input_convenience.html>`_.
